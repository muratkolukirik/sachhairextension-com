# sachhairextension.com

Static marketing site for **Sach & Vogue Hair** — premium human hair extensions manufacturer in Fethiye / Ölüdeniz, Türkiye.

## Commands

| Command           | Action                          |
| :---------------- | :------------------------------ |
| `npm install`     | Install dependencies            |
| `npm run dev`     | Start dev server                |
| `npm run build`   | Build to `./dist/`              |
| `npm run preview` | Preview production build        |

## Deploy

Built for Cloudflare Pages. Production URL: https://sachhairextension.com
